# 64.最小路径和

## 题目描述

给定一个包含非负整数的 `*m* x *n*` 网格 `grid` ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

**说明：**每次只能向下或者向右移动一步。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/11/05/minpath.jpg)

```
输入：grid = [[1,3,1],[1,5,1],[4,2,1]]
输出：7
解释：因为路径 1→3→1→1→1 的总和最小。
```

**示例 2：**

```
输入：grid = [[1,2,3],[4,5,6]]
输出：12
```

 

**提示：**

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 200`
- `0 <= grid[i][j] <= 200`



## 题解

~~~
class Solution {
public:
    int minPathSum(vector<vector<int>>& grid) {
        vector<vector<int>> distance(grid.size(),vector<int>(grid[0].size(),INT_MAX));
        int result = 0;
        distance[0][0] = grid[0][0];
        for(int i=0;i<grid.size();i++)
        {
            for(int j=0;j<grid[0].size();j++)
            {
                if(i+1<grid.size())
                {
                    distance[i+1][j] = min(distance[i+1][j],distance[i][j]+grid[i+1][j]);
                }
                if(j+1<grid[0].size())
                {
                    distance[i][j+1] = min(distance[i][j+1],distance[i][j]+grid[i][j+1]);
                }
            }
        }
        return distance[grid.size()-1][grid[0].size()-1];
    }
};
~~~
