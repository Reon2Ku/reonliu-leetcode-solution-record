# 1317.将证书转换为两个无零整数的和

## 题目描述

「无零整数」是十进制表示中 **不含任何 0** 的正整数。

给你一个整数 `n`，请你返回一个 **由两个整数组成的列表** `[a, b]`，满足：

- `a` 和 `b` 都是无零整数
- `a + b = n`

题目数据保证至少有一个有效的解决方案。

如果存在多个有效解决方案，你可以返回其中任意一个。

 

**示例 1：**

```
输入：n = 2
输出：[1,1]
解释：a = 1, b = 1。a + b = n 并且 a 和 b 的十进制表示形式都不包含任何 0。
```

**示例 2：**

```
输入：n = 11
输出：[2,9]
```

**示例 3：**

```
输入：n = 10000
输出：[1,9999]
```

**示例 4：**

```
输入：n = 69
输出：[1,68]
```

**示例 5：**

```
输入：n = 1010
输出：[11,999]
```

 

**提示：**

- `2 <= n <= 104`



## 题解

捏吗，你干嘛搞这么复杂，傻逼

~~~
class Solution {
public:
    int haszero(int a)
    {
        int count = -1;
        while(a>0)
        {
            count++;
            if(a%10==0) return count;
            a/=10;
        }
        return 0;
    }
    vector<int> getNoZeroIntegers(int n) {
        vector<int> func(10,1);
        func[1] = 2;
        vector<int> result;
        int temp = n;
        while(temp>1)
        {
            result.push_back(func[temp%10]);
            temp/=10;
        }
        int a = 0;
        for(int i=result.size()-1;i>=0;i--)
        {
            a*=10;
            //cout<<result[i]<<endl;
            a+=result[i];
        }
        cout<<a<<endl;

        while(int count = haszero(n-a)>0)
        {
            a+=pow(10,count);
        } 
        return {a,n-a};
    }
};
~~~

class Solution {
public:
    bool haszero(int a)
    {
        while(a>0)
        {
            if(a%10==0) return true;
            a/=10;
        }
        return false;
    }
    vector<int> getNoZeroIntegers(int n) {
        int a = 1;
        int b = n-a;
        while(haszero(a)||haszero(b))
        {
            a++;
            b--;
        }
        return {a,b};
    }
};
