# 5.最长回文子串

## 题目描述

给你一个字符串 `s`，找到 `s` 中最长的 回文 子串。

 

**示例 1：**

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

**示例 2：**

```
输入：s = "cbbd"
输出："bb"
```

 

**提示：**

- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母组成



## 题解

多亏了现场提示，但为什么我写的已经是O(n^2)了还是只能击败50%，花了120ms？

那没事了，标准答案居然花了360ms，感觉是我家网络问题

~~~
class Solution {
public:
    string longestPalindrome(string s) {
        int size = s.size();
        int length = 1;
        int start = 0;
        vector<vector<bool>> isPalindrome(size,vector<bool>(size,0));
        for(int i=0;i<size;i++) isPalindrome[i][i] = 1;
        for(int i=1;i<size;i++) 
        {
            for(int j=0;j+i<size;j++)
            {
                if(s[j]!=s[i+j]) continue;
                else if(i==1) 
                {
                    if(length==1) 
                    {
                        start = j;
                        length = 2;
                    }
                    isPalindrome[j][j+i] = 1;
                }
                else if(isPalindrome[j+1][j+i-1])
                {
                    isPalindrome[j][i+j] = 1;
                    if(i+1>length) 
                    {
                        length = i+1;
                        start = j;
                    }
                }
            }
        }
        return s.substr(start,length);
    }
};
~~~
