# 96.不同的二叉搜索树

给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2021/01/18/uniquebstn3.jpg)

```
输入：n = 3
输出：5
```

**示例 2：**

```
输入：n = 1
输出：1
```



## 题解

自己想的，我真棒，虽然云里雾里做对了就是了

~~~
class Solution {
public:
    int numTrees(int n) {
        if(n<=2) return n;
        vector<int> result(n+2,0);
        result[0] = 1;
        result[1] = 1;
        for(int i=2;i<=n+1;i++)
        {
            for(int j=0;j<i;j++)
            {
                result[i] += result[j] * result[i-j];
            }
        }
        return result[result.size()-1];
    }
};
~~~
