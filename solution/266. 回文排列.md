# 266. 回文排列

## question

Given a string `s`, return `true` *if a permutation of the string could form a* ***palindrome*** *and* `false` *otherwise*.

 

**Example 1:**

```
Input: s = "code"
Output: false
```

**Example 2:**

```
Input: s = "aab"
Output: true
```

**Example 3:**

```
Input: s = "carerac"
Output: true
```

 

**Constraints:**

- `1 <= s.length <= 5000`
- `s` consists of only lowercase English letters.



## solution

时间100，空间50

~~~
class Solution {
public:
    bool canPermutePalindrome(string s) {
        vector<int> record(26,0);
        for(char c:s)
        {
            record[c-'a']++;
        }
        int temp = 0;
        for(int i:record)
        {
            temp+=i%2;
            if(temp>1) return false;
        }
        return true;
    }
};
~~~
