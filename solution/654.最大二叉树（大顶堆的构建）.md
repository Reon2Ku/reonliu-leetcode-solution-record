# 654.最大二叉树（大顶堆的构建）

## 题目描述

给定一个不重复的整数数组 `nums` 。 **最大二叉树** 可以用下面的算法从 `nums` 递归地构建:

1. 创建一个根节点，其值为 `nums` 中的最大值。
2. 递归地在最大值 **左边** 的 **子数组前缀上** 构建左子树。
3. 递归地在最大值 **右边** 的 **子数组后缀上** 构建右子树。

返回 *`nums` 构建的* ***最大二叉树\*** 。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/12/24/tree1.jpg)

```
输入：nums = [3,2,1,6,0,5]
输出：[6,3,5,null,2,0,null,null,1]
解释：递归调用如下所示：
- [3,2,1,6,0,5] 中的最大值是 6 ，左边部分是 [3,2,1] ，右边部分是 [0,5] 。
    - [3,2,1] 中的最大值是 3 ，左边部分是 [] ，右边部分是 [2,1] 。
        - 空数组，无子节点。
        - [2,1] 中的最大值是 2 ，左边部分是 [] ，右边部分是 [1] 。
            - 空数组，无子节点。
            - 只有一个元素，所以子节点是一个值为 1 的节点。
    - [0,5] 中的最大值是 5 ，左边部分是 [0] ，右边部分是 [] 。
        - 只有一个元素，所以子节点是一个值为 0 的节点。
        - 空数组，无子节点。
```

## 题解

延续106的思路，非常容易写出

~~~
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    TreeNode* construct(vector<int>& nums)
    {
        if(nums.size()==0) return NULL;

        int maxIndex = 0;
        for(int i=0;i<nums.size();i++)
        {
            if(nums[i]>=nums[maxIndex])
            {
                maxIndex = i;
            }
        }

        int rootValue = nums[maxIndex];
        TreeNode* root = new TreeNode(rootValue);

        if(nums.size()==1) return root;

        vector<int> leftNum(nums.begin(),nums.begin()+maxIndex);
        vector<int> rightNum(nums.begin()+maxIndex+1,nums.end());

        root->left = construct(leftNum);
        root->right = construct(rightNum);

        return root;
    }
    TreeNode* constructMaximumBinaryTree(vector<int>& nums) {
        return construct(nums);
    }
};
~~~

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    TreeNode* construct(vector<int>& nums,int nbegin,int nend)
    {
        if(nend-nbegin==0) return NULL;

        int maxIndex = nbegin;
        for(int i=nbegin;i<nend;i++)
            if(nums[i]>=nums[maxIndex]) maxIndex = i;

        int rootValue = nums[maxIndex];
        TreeNode* root = new TreeNode(rootValue);

        if(nend-nbegin==1) return root;

        //左闭右开
        int leftbegin = nbegin;
        int leftend = maxIndex;
        int rightbegin = maxIndex+1;
        int rightend = nend;

        root->left = construct(nums,leftbegin,leftend);
        root->right = construct(nums,rightbegin,rightend);

        return root;
    }
    TreeNode* constructMaximumBinaryTree(vector<int>& nums) {
        return construct(nums,0,nums.size());
    }
};
