# 77.组合（回溯算法经典例题）

## 题目描述

给定两个整数 `n` 和 `k`，返回范围 `[1, n]` 中所有可能的 `k` 个数的组合。

你可以按 **任何顺序** 返回答案。

 

**示例 1：**

```
输入：n = 4, k = 2
输出：
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```

**示例 2：**

```
输入：n = 1, k = 1
输出：[[1]]
```

先看看回溯算法的经典模板吧：

要是没有思路了就来看看这个模板

~~~
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
~~~

接下来是这道题的题解：

~~~
class Solution {
public:
    vector<vector<int>> result;
    vector<int> path;
    void combineRec(int startIndex, int n, int k)
    {
        if(path.size() == k)
        {
            result.push_back(path);
            return;
        }
        for(int i=startIndex;i<=n;i++)
        {
            path.push_back(i);
            combineRec(i+1,n,k);
            path.pop_back();
        }

    }
    vector<vector<int>> combine(int n, int k) {
        combineRec(1,n,k);
        return result;
    }
};
~~~

剪枝优化：for循环的终止条件还可以再多考量一下：

~~~
class Solution {
public:
    vector<vector<int>> result;
    vector<int> path;
    void combineRec(int startIndex, int n, int k)
    {
        if(path.size() == k)
        {
            result.push_back(path);
            return;
        }
        for(int i=startIndex;i<=n-k+path.size()+1;i++)
        {
            path.push_back(i);
            combineRec(i+1,n,k);
            path.pop_back();
        }

    }
    vector<vector<int>> combine(int n, int k) {
        combineRec(1,n,k);
        return result;
    }
};
~~~
