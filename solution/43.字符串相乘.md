# 43.字符串相乘

## 题目描述

给定两个以字符串形式表示的非负整数 `num1` 和 `num2`，返回 `num1` 和 `num2` 的乘积，它们的乘积也表示为字符串形式。

**注意：**不能使用任何内置的 BigInteger 库或直接将输入转换为整数。

 

**示例 1:**

```
输入: num1 = "2", num2 = "3"
输出: "6"
```

**示例 2:**

```
输入: num1 = "123", num2 = "456"
输出: "56088"
```

 

**提示：**

- `1 <= num1.length, num2.length <= 200`
- `num1` 和 `num2` 只能由数字组成。
- `num1` 和 `num2` 都不包含任何前导零，除了数字0本身。



## 题解

我的方法就是最权威的

速度击败74%，内存击败98%

~~~
class Solution {
public:
    string multiply(string num1, string num2) {
        if(num1=="0" || num2=="0") return "0";
        int s1 = num1.size();
        int s2 = num2.size();
        int size = s1 + s2;
        string result(size,'0');
        for(int i=s1-1;i>=0;i--)
        {
            for(int j=s2-1;j>=0;j--)
            {
                int n1 = (int)(num1[i]-'0');
                int n2 = (int)(num2[j]-'0');
                int mul = n1*n2;
                // int indexShift = s1+s2-2-i-j;
                // int index = size-1-indexShift;
                int index = i+j+1;
                int curr = (int)(result[index-1]-'0');
                curr*=10;
                curr+=(int)(result[index]-'0');
                curr+=mul;
                result[index] = (char)(curr%10) + '0';
                if(curr>=10) result[index-1] = (char)(curr/10) + '0';
                //cout<<mul<<endl;
                //cout<<result<<endl;
            }
        }
        //cout<<result<<endl;
        if(result[0]=='0') result = result.substr(1);
        return result;
    }
};
~~~
