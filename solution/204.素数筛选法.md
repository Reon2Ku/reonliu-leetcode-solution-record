# 204.素数筛选法

## 题目描述

给定整数 `n` ，返回 *所有小于非负整数 `n` 的质数的数量* 。

 

**示例 1：**

```
输入：n = 10
输出：4
解释：小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。
```

**示例 2：**

```
输入：n = 0
输出：0
```

**示例 3：**

```
输入：n = 1
输出：0
```

 

**提示：**

- `0 <= n <= 5 * 106`



## 题解

经典的素数筛选法，仅看了思路，代码复现是自己做的。

~~~
class Solution {
public:
    int countPrimes(int n) {
        if(n<=2) return 0;
        vector<bool> is_prime(n,true);
        is_prime[0]=false;
        is_prime[1]=false;
        for(int i=2;i<n;i++)
        {
            if(!is_prime[i]) continue;
            int curr = i;
            while(true)
            {
                curr+=i;
                if(curr<n)
                {
                    is_prime[curr]=false;
                }
                else break;
            }
        }
        int result = 0;
        for(int i=2;i<n;i++)
        {
            if(is_prime[i]) result++;
        }
        return result;
        //如果要你求m到n之间的素数，那就把最后的i初始化为m就行了

    }
};
~~~
