# 19.删除链表的倒数第N个节点

## 题目描述

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg)

```
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
```

**示例 2：**

```
输入：head = [1], n = 1
输出：[]
```

**示例 3：**

```
输入：head = [1,2], n = 1
输出：[1]
```

## 题解

初见白痴解法：很多地方都不严谨，但简单粗暴

~~~
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode *temp = head;
        int size=0;
        while(temp!=nullptr)
        {
            temp = temp->next;
            size++;
        }
        if(n>size)
            return head;
        else if (n == size)
        {
            ListNode* result = head->next;
            delete head;
            return result;
        }
        else
        {
            ListNode *curr = head;
            for(int i=0;i<size-n;i++)
            {
                temp = curr;
                curr = curr->next;
            }
            temp->next = curr->next;
            delete curr;
            return head;
        }
    }
};O(2n)
~~~

二见进阶解法：只遍历一遍降低复杂度

~~~
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
            ListNode *dummyHead = new ListNode(0);
            dummyHead->next = head;
            ListNode *curr = dummyHead;
            ListNode *temp = curr;
            for(int i=0;i<n;i++)
            {
                curr = curr->next;
            }
            while(curr->next!=nullptr)
            {
                temp = temp->next;
                curr = curr->next;
            }
            temp->next = temp->next->next;
            ListNode *result = dummyHead->next;
            delete dummyHead;
            return result;
        }
    
};O(n)
~~~
