# 977.有序数组的平方

## 题目描述

给你一个按 **非递减顺序** 排序的整数数组 `nums`，返回 **每个数字的平方** 组成的新数组，要求也按 **非递减顺序** 排序。



 

**示例 1：**

```
输入：nums = [-4,-1,0,3,10]
输出：[0,1,9,16,100]
解释：平方后，数组变为 [16,1,0,9,100]
排序后，数组变为 [0,1,9,16,100]
```

**示例 2：**

```
输入：nums = [-7,-3,2,3,11]
输出：[4,9,9,49,121]
```

 

**提示：**

- `1 <= nums.length <= 104`
- `-104 <= nums[i] <= 104`
- `nums` 已按 **非递减顺序** 排序

 

**进阶：**

- 请你设计时间复杂度为 `O(n)` 的算法解决本问题



## 题解

`````
//自己写的弱智暴力解，时间复杂度可悲的来到了 O(n + nlogn)
class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        for(int i=0;i<nums.size();i++)
        {
            nums[i]=nums[i]*nums[i];
        }
        sort(nums.begin(),nums.end());
        return nums;
    }
};
//先平方再排序...
//我的评价是，不如调用一个push_back再调用一个sort，两个库函数秒了
`````

还是看看复杂度O(n)的解法吧

`````
class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        vector<int> temp;
        temp.resize(nums.size());
        int i=0;
        int j=nums.size()-1;
        for(int count=nums.size()-1;count>=0;count--)
        {
            if((nums[i]*nums[i]) <= (nums[j]*nums[j]))
            {
                temp[count]=nums[j]*nums[j];
                j--;
            }
            else{
                temp[count]=nums[i]*nums[i];
                i++;
            }
        }
        return temp;
    }
};
//虽然也是自己写的但思想和标准答案比较像了。
`````

标准答案：

`````
class Solution {
public:
    vector<int> sortedSquares(vector<int>& A) {
        int k = A.size() - 1;
        vector<int> result(A.size(), 0);
        for (int i = 0, j = A.size() - 1; i <= j;) { // 注意这里要i <= j，因为最后要处理两个元素
            if (A[i] * A[i] < A[j] * A[j])  {
                result[k--] = A[j] * A[j];
                j--;
            }
            else {
                result[k--] = A[i] * A[i];
                i++;
            }
        }
        return result;
    }
};
`````
