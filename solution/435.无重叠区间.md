# 435.无重叠区间

给定一个区间的集合 `intervals` ，其中 `intervals[i] = [starti, endi]` 。返回 *需要移除区间的最小数量，使剩余区间互不重叠* 。

**注意** 只在一点上接触的区间是 **不重叠的**。例如 `[1, 2]` 和 `[2, 3]` 是不重叠的。

 

**示例 1:**

```
输入: intervals = [[1,2],[2,3],[3,4],[1,3]]
输出: 1
解释: 移除 [1,3] 后，剩下的区间没有重叠。
```

**示例 2:**

```
输入: intervals = [ [1,2], [1,2], [1,2] ]
输出: 2
解释: 你需要移除两个 [1,2] 来使剩下的区间没有重叠。
```

**示例 3:**

```
输入: intervals = [ [1,2], [2,3] ]
输出: 0
解释: 你不需要移除任何区间，因为它们已经是无重叠的了。
```





## 题解

和上一题452解题思路过于相像以至于秒了

~~~
class Solution {
public:
    int eraseOverlapIntervals(vector<vector<int>>& intervals) {
        if(intervals.size()<=1) return 0;
        int column = 0;
        sort(intervals.begin(),intervals.end(),[column](const vector<int>& a, const vector<int>& b)
        {
            if(a[0]==b[0]) return a[1]<b[1];
            return a[0]<b[0];
        });
        vector<int> curr = intervals[0];
        int result = 0;
        //cout<<curr[0]<<"  "<<curr[1]<<endl;
        for(int i=1;i<intervals.size();i++)
        {
            //cout<<intervals[i][0]<<"  "<<intervals[i][1]<<endl;
            if(intervals[i][0]<curr[1]) 
            {
                //cout<<"result++ at:"<<intervals[i][0]<<"  "<<intervals[i][1]<<endl;
                curr[0] = intervals[i][0];
                if(intervals[i][1]<curr[1]) curr[1] = intervals[i][1];
                result++;
            }
            else curr = intervals[i];
        }
        return result;
    }
};
~~~
