# 18.四数之和（有空可以重做，思路与三数之和相似，但用例更刁钻）

## 题目描述

给你一个由 `n` 个整数组成的数组 `nums` ，和一个目标值 `target` 。请你找出并返回满足下述全部条件且**不重复**的四元组 `[nums[a], nums[b], nums[c], nums[d]]` （若两个四元组元素一一对应，则认为两个四元组重复）：

- `0 <= a, b, c, d < n`
- `a`、`b`、`c` 和 `d` **互不相同**
- `nums[a] + nums[b] + nums[c] + nums[d] == target`

你可以按 **任意顺序** 返回答案 。

 

**示例 1：**

```
输入：nums = [1,0,-1,0,-2,2], target = 0
输出：[[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]
```

**示例 2：**

```
输入：nums = [2,2,2,2,2], target = 8
输出：[[2,2,2,2]]
```

 **提示：**

- `1 <= nums.length <= 200`
- `-109 <= nums[i] <= 109`
- `-109 <= target <= 109`

## 题解

有了三数之和的经验，做这道题压力小了很多，思路是差不多的只是多了一层for循环而已，但是leetcode在测试用例的设计上面非常刁钻，导致我修修改改了好久。

~~~
class Solution {
public:
    vector<vector<int>> fourSum(vector<int>& nums, int target) {
        vector<vector<int>> result;
        if(nums.size()<4) return result;//刁钻处一
        sort(nums.begin(),nums.end());
        for(int i=0;i<nums.size()-3;i++)
        {
            if(nums[i]>target&&nums[i]>=0) break;//刁钻二
            if(i>0 && nums[i] == nums[i-1]) continue;
            for(int j=nums.size()-1;j>i+2;j--)
            {
                if(nums[j]+nums[i]>target&&nums[i]>0) break;//刁钻三
                if(nums[j]+nums[i]<target&&nums[j]<0) break;//刁钻四
                if(j<nums.size()-1 && nums[j]==nums[j+1]) continue;
                int left = i+1;
                int right =j-1;
                while(right>left)
                {
                    if(nums[i]+nums[j]+nums[left]+nums[right]>target) right--;
                    else if(nums[i]+nums[j]+nums[left]+nums[right]<target) left++;
                    else{
                    result.push_back({nums[i],nums[left],nums[right],nums[j]});
                    while(right>left && nums[right]==nums[right-1]) right--;
                    while(right>left && nums[left]==nums[left+1]) left++;
                    right--;
                    left++;
                    }
                }
            }
        }
        return result;
    }
};
~~~

标注刁钻的四行代码花了我将近一个小时，其他代码有之前的思路十分钟左右就敲出来了
