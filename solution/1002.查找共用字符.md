# 1002.查找共用字符

## 题目描述

给你一个字符串数组 `words` ，请你找出所有在 `words` 的每个字符串中都出现的共用字符（**包括重复字符**），并以数组形式返回。你可以按 **任意顺序** 返回答案。

 

**示例 1：**

```
输入：words = ["bella","label","roller"]
输出：["e","l","l"]
```

**示例 2：**

```
输入：words = ["cool","lock","cook"]
输出：["c","o"]
```

事后在leetcode上搜了这道题，还是没有很好的思路，看了一眼题解的思路最后自己写出来了。

~~~
class Solution {
public:
    vector<string> commonChars(vector<string>& words) {
        vector<int> result;
        vector<int> minValue;
        result.resize(26,0);
        minValue.resize(26,0);
        for(int i=0;i<words[0].size();i++)
        {
            minValue[words[0][i]-'a']++;
        }
        for(int i=1;i<words.size();i++)
        {
            for(int j=0;j<words[i].size();j++)
            {
                result[words[i][j]-'a']++;
            }
            for(int j=0;j<26;j++)
            {
                if(result[j]<minValue[j])
                {
                    minValue[j]=result[j];
                }
            }
            fill(result.begin(),result.end(),0);
        }
        vector<string> myResult;
        for(int i=0;i<26;i++)
        {
            if(minValue[i]>0)
            {
                for(int j=0;j<minValue[i];j++)
                {
                    myResult.push_back(string(1,char('a'+i)));
                }
            }
        }
        return myResult;
    }
};
~~~
