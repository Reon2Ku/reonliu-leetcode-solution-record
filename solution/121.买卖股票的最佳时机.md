# 121.买卖股票的最佳时机

## 题目描述

给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

 

**示例 1：**

```
输入：[7,1,5,3,6,4]
输出：5
解释：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。
```

**示例 2：**

```
输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下, 没有交易完成, 所以最大利润为 0。
```



## 题解

简单题姑且还是自己写的出来，不过写的很是没有章法，后续进阶题可能找不到思路

也不算没有章法吧，你的解法是贪心而非动规

~~~
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        //if(prices.size()==1) return 0;
        int result = 0;
        int buyinprice = INT_MAX;
        for(int i=0;i<prices.size();i++)
        {
            if(prices[i]-buyinprice < 0) 
            {
                buyinprice = prices[i];
                continue;
            }
            else
            {
                result = max(result,prices[i]-buyinprice);
            }
        }
        return result;
    }
};
~~~

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int buyinprice = INT_MAX;
        vector<int> dp(prices.size()+1,0);
        if(prices.size()<=1) return 0;
        for(int i=1;i<=prices.size();i++)
        {
            //cout<<dp[i]<<endl;
            if(prices[i-1]-buyinprice < 0) 
            {
                buyinprice = prices[i-1];
                dp[i] = dp[i-1];
                continue;
            }
            else
            {
                dp[i] = max(dp[i-1],prices[i-1]-buyinprice);
            }
        }
        return dp[prices.size()];
    }
};

为什么你会觉得自己在强行用dp？因为你用的根本不是dp！

看看，这才是真正的dp

~~~
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        vector<vector<int>> dp(prices.size(),vector<int>(2,0));
        dp[0][0] = 0-prices[0];
        dp[0][1] = 0;
        for(int i=1;i<prices.size();i++)
        {
            dp[i][0] = max(dp[i-1][0],0-prices[i]);
            dp[i][1] = max(dp[i-1][1],dp[i-1][0]+prices[i]); 
        }
        return dp[prices.size()-1][1];
    }
};
~~~
