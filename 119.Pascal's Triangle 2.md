# 119.Pascal's Triangle 2

## solution

从官方题解看了下思路后，自己敲的。性能明显薄纱自己提交的第一版

~~~
class Solution {
public:
    vector<int> getRow(int rowIndex) {
        vector<int> vec(rowIndex+1);//最终结果中的行包含rowIndex+1个元素
        vec[0] = 1;
        int origin = rowIndex;
        while(rowIndex--)
        {
            for(int i=origin;i>0;i--)
            {
                vec[i] = vec[i]+vec[i-1];
            }
        }
        return vec;

    }
};
~~~
