# |=计算和&=计算

| **运算符** | **功能**         | **运算规则**                           | **典型用途**           |
| :--------- | :--------------- | :------------------------------------- | :--------------------- |
| **`\|=`**  | 设置特定位为 `1` | 对应位有 `1`则结果位为 `1`             | 启用标志位、添加权限   |
| **`&=`**   | 清除特定位为 `0` | 对应位全 `1`结果位才为 `1`，否则为 `0` | 禁用标志位、屏蔽无关位 |

| **运算符** | **符号** | **功能**                       | **常见用途**                 |
| :--------- | :------- | :----------------------------- | :--------------------------- |
| 按位与     | `&`      | 两位均为 `1`时结果为 `1`       | 掩码操作、判断奇偶、清零位   |
| 按位或     | \|       | \|                             | 两位有 `1`时结果为 `1`       |
| 按位取反   | `~`      | 所有位取反                     | 生成掩码、配合其他位运算使用 |
| 左移       | `<<`     | 左移 `k`位，低位补 `0`         | 快速乘以 2*k*                |
| 右移       | `>>`     | 右移 `k`位，高位补 `0`或符号位 | 快速除以 2*k*                |

^=是异或计算

同或：

~~~
int xnor(int a, int b) {
    return ~(a ^ b);  // 先异或再取反
}
~~~

左移右移

~~~
unsigned u1 = 0x0F;       // 二进制: 0000 1111 (15)
u1 = u1 << 4;             // 左移 4 位 → 1111 0000 (240)
cout << "u1 << 4 = " << u1 << endl; // 输出 240

// 右移示例（无符号）
unsigned u2 = 0xF0;      // 二进制: 1111 0000 (240)
u2 = u2 >> 4;            // 右移 4 位 → 0000 1111 (15)
cout << "u2 >> 4 = " << u2 << endl; // 输出 15
~~~
